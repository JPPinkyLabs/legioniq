# LegionIQ - AI-Powered Gaming Assistant

## About

LegionIQ is an AI-powered gaming assistance platform that provides intelligent recommendations based on game screenshots. Users can upload screenshots and receive expert advice across three categories: gameplay help, technical support, and strategic guidance.

## Features

### Core Functionality

- **Screenshot Upload**: Upload game screenshots for instant analysis
- **OCR Text Extraction**: Automatic text recognition from screenshots using OCR.space API
- **AI-Powered Analysis**: Advanced analysis powered by OpenAI GPT-4o
- **Category-Based Assistance**: Three specialized categories:
  - Gameplay Help: Mechanics, controls, and gameplay strategies
  - Technical Support: Bug troubleshooting and performance issues
  - Strategic Advice: High-level strategy and decision-making guidance

### User Experience

- **Conversational Interface**: Chat-based interaction with AI assistant
- **Request History**: View and manage all previous analysis requests
- **Request Details**: Detailed view of individual requests with full AI responses
- **Rating System**: Rate AI responses to help improve service quality
- **User Account Management**: Secure authentication and account settings
- **Dark/Light Theme**: Toggle between dark and light themes

### Platform Features

- **Secure Authentication**: User accounts with session management via Supabase
- **Privacy Policy**: Complete privacy policy and terms of service
- **Responsive Design**: Works on desktop and mobile devices
- **Real-time Updates**: Live conversation updates during AI analysis

## Technology Stack

- **Frontend**: React + Vite + TypeScript + Tailwind CSS + ShadCN UI
- **Backend**: Supabase with Edge Functions
- **AI/ML**: OpenAI GPT-4o API
- **OCR**: OCR.space API
- **Database**: PostgreSQL (via Supabase)
- **Authentication**: Supabase Authentication
- **State Management**: React Query (TanStack Query) + Zustand
- **Routing**: React Router
